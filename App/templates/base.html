Telenor Compute Visualizer

Welcome to Telenor Computing Machine Visualizer, this application is a result of Summer Internship Program of Umaid Zaffar in 2020.
The application will allow you to visualize each instance of your compute and notify any overlapping sets.

The backend was developed using Python Flask Framework
The frontend was developed using Jinja2/HTML/CSS
Prototyped in Jupyter Notebooks
Libraries: Pandas, Numpy, Matplotlib, CanvasJS

Steps:
  1. Select the Zone.
  2. Select the Computing Machine.
  3. Visualize the latest data.
